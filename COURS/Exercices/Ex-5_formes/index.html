"Calcul de forme géometrique"
Réalisez une page web qui permet de calculer l'aire et le périmètre de forme.
Les formes géométriques sont sélectionnables via une balise "select" (menu déroulant) et les choix possibles sont :

  • Rectangle
  • Triangle Rectangle
  • Carré
  • Cercle

Les valeurs à encoder par l'utilisateur pour les différents choix sont :
  • Rectangle : Hauteur et Largeur
  • Triangle : Base et Hauteur
  • Carré : Un Coté
  • Cercle : Le Rayon

L'utilisateur peut encoder les valeurs via des inputs de type texte.
Bonus : Affiche le nombre d'input adapté au choix de l'utilisateur.
